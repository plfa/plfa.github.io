Introduction

In this chapter we prove that beta reduction is confluent, a property also known as Church-Rosser. That is, if there are reduction sequences from any term L to two different terms M‚ÇÅ and M‚ÇÇ, then there exist reduction sequences from those two terms to some common term N. In pictures:

    L
   / \
  /   \
 /     \
M‚ÇÅ      M‚ÇÇ
 \     /
  \   /
   \ /
    N

where downward lines are instances of ‚Äî‚Ü†.

Confluence is studied in many other kinds of rewrite systems besides the lambda calculus, and it is well known how to prove confluence in rewrite systems that enjoy the diamond property, a single-step version of confluence. Let ‚áõ be a relation. Then ‚áõ has the diamond property if whenever L ‚áõ M‚ÇÅ and L ‚áõ M‚ÇÇ, then there exists an N such that M‚ÇÅ ‚áõ N and M‚ÇÇ ‚áõ N. This is just an instance of the same picture above, where downward lines are now instance of ‚áõ. If we write ‚áõ* for the reflexive and transitive closure of ‚áõ, then confluence of ‚áõ* follows immediately from the diamond property.

Unfortunately, reduction in the lambda calculus does not satisfy the diamond property. Here is a counter example.

(Œª x. x x)((Œª x. x) a) ‚Äî‚Üí (Œª x. x x) a
(Œª x. x x)((Œª x. x) a) ‚Äî‚Üí ((Œª x. x) a) ((Œª x. x) a)

Both terms can reduce to a a, but the second term requires two steps to get there, not one.

To side-step this problem, we‚Äôll define an auxilliary reduction relation, called parallel reduction, that can perform many reductions simultaneously and thereby satisfy the diamond property. Furthermore, we show that a parallel reduction sequence exists between any two terms if and only if a beta reduction sequence exists between them. Thus, we can reduce the proof of confluence for beta reduction to confluence for parallel reduction.

The parallel reduction relation is defined as follows.

    ‚Üí ∆õ N ‚áõ ∆õ N‚Ä≤

We remark that the pabs, papp, and pbeta rules perform reduction on all their subexpressions simultaneously. Also, the pabs rule is akin to the Œ∂ rule and pbeta is akin to Œ≤.

Equivalence between parallel reduction and reduction

Here we prove that for any M and N, M ‚áõ* N if and only if M ‚Äî‚Ü† N. The only-if direction is particularly easy. We start by showing that if M ‚Äî‚Üí N, then M ‚áõ N. The proof is by induction on the reduction M ‚Äî‚Üí N.

beta-par {Œì} {‚òÖ} {∆õ N} (Œ∂ r) = pabs (beta-par r)

With this lemma in hand we complete the only-if direction, that M ‚Äî‚Ü† N implies M ‚áõ* N. The proof is a straightforward induction on the reduction sequence M ‚Äî‚Ü† N.

   L ‚áõ‚ü® beta-par b ‚ü© betas-pars bs

Now for the other direction, that M ‚áõ* N implies M ‚Äî‚Ü† N. The proof of this direction is a bit different because it‚Äôs not the case that M ‚áõ N implies M ‚Äî‚Üí N. After all, M ‚áõ N performs many reductions. So instead we shall prove that M ‚áõ N implies M ‚Äî‚Ü† N.

The proof is by induction on M ‚áõ N.

Substitution lemma for parallel reduction

Our next goal is the prove the diamond property for parallel reduction. But to do that, we need to prove that substitution respects parallel reduction. That is, if N ‚áõ N‚Ä≤ and M ‚áõ M‚Ä≤, then N [ M ] ‚áõ N‚Ä≤ [ M‚Ä≤ ]. We cannot prove this directly by induction, so we generalize it to: if N ‚áõ N‚Ä≤ and the substitution œÉ pointwise parallel reduces to œÑ, then subst œÉ N ‚áõ subst œÑ N‚Ä≤. We define the notion of pointwise parallel reduction as follows.

par-subst : ‚àÄ{Œì Œî} ‚Üí Subst Œì Œî ‚Üí Subst Œì Œî ‚Üí Set
par-subst {Œì}{Œî} œÉ œÉ‚Ä≤ = ‚àÄ{A}{x : Œì ‚àã A} ‚Üí œÉ x ‚áõ œÉ‚Ä≤ x

Because substitution depends on the extension function exts, which in turn relies on rename, we start with a version of the substitution lemma, called par-rename, that is specialized to renamings. The proof of par-rename relies on the fact that renaming and substitution commute with one another, which is a lemma that we import from Chapter Substitution and restate here.

rename-subst-commute : ‚àÄ{Œì Œî}{N : Œì , ‚òÖ ‚ä¢ ‚òÖ}{M : Œì ‚ä¢ ‚òÖ}{œÅ : Rename Œì Œî }
    ‚Üí (rename (ext œÅ) N) [ rename œÅ M ] ‚â° rename œÅ (N [ M ])
rename-subst-commute {N = N} = plfa.part2.Substitution.rename-subst-commute {N = N}

Now for the par-rename lemma.

par-rename : ‚àÄ{Œì Œî A} {œÅ : Rename Œì Œî} {M M‚Ä≤ : Œì ‚ä¢ A}
  ‚Üí M ‚áõ M‚Ä≤
    ------------------------
  ‚Üí rename œÅ M ‚áõ rename œÅ M‚Ä≤

par-rename (pabs p) = pabs (par-rename p)
par-rename (papp p‚ÇÅ p‚ÇÇ) = papp (par-rename p‚ÇÅ) (par-rename p‚ÇÇ)
par-rename {Œì}{Œî}{A}{œÅ} (pbeta{Œì}{N}{N‚Ä≤}{M}{M‚Ä≤} p‚ÇÅ p‚ÇÇ)
    with pbeta (par-rename{œÅ = ext œÅ} p‚ÇÅ) (par-rename{œÅ = œÅ} p‚ÇÇ)
... | G rewrite rename-subst-commute{Œì}{Œî}{N‚Ä≤}{M‚Ä≤}{œÅ} = G

The proof is by induction on M ‚áõ M‚Ä≤. The first four cases are straightforward so we just consider the last one for pbeta.

  • Suppose (∆õ N) ¬∑ M ‚áõ N‚Ä≤ [ M‚Ä≤ ] because N ‚áõ N‚Ä≤ and M ‚áõ M‚Ä≤. By the induction hypothesis, we have rename (ext œÅ) N ‚áõ rename (ext œÅ) N‚Ä≤ and rename œÅ M ‚áõ rename œÅ M‚Ä≤. So by pbeta we have (∆õ rename (ext œÅ) N) ¬∑ (rename œÅ M) ‚áõ (rename (ext œÅ) N) [ rename œÅ M ]. However, to conclude we instead need parallel reduction to rename œÅ (N [ M ]). But thankfully, renaming and substitution commute with one another.

With the par-rename lemma in hand, it is straightforward to show that extending substitutions preserves the pointwise parallel reduction relation.

par-subst-exts : ‚àÄ{Œì Œî} {œÉ œÑ : Subst Œì Œî}
  ‚Üí par-subst œÉ œÑ
    ------------------------------------------
  ‚Üí ‚àÄ{B} ‚Üí par-subst (exts œÉ {B = B}) (exts œÑ)

The next lemma that we need for proving that substitution respects parallel reduction is the following which states that simultaneoous substitution commutes with single substitution. We import this lemma from Chapter Substitution and restate it below.

We are ready to prove that substitution respects parallel reduction.

  ‚Üí par-subst œÉ œÑ  ‚Üí  M ‚áõ M‚Ä≤

  ‚Üí subst œÉ M ‚áõ subst œÑ M‚Ä≤

We proceed by induction on M ‚áõ M‚Ä≤.

  • Suppose L ¬∑ M ‚áõ L‚Ä≤ ¬∑ M‚Ä≤ because L ‚áõ L‚Ä≤ and M ‚áõ M‚Ä≤. By the induction hypothesis we have subst œÉ L ‚áõ subst œÑ L‚Ä≤ and subst œÉ M ‚áõ subst œÑ M‚Ä≤, so we conclude by rule papp.

  • Suppose (∆õ N) ¬∑ M ‚áõ N‚Ä≤ [ M‚Ä≤ ] because N ‚áõ N‚Ä≤ and M ‚áõ M‚Ä≤. Again we obtain par-subst (exts œÉ) (exts œÑ) by par-subst-exts.       (subst (exts œÉ) N) [ subst œÉ M ] ‚â° subst œÉ (N [ M ])

    So we have parallel reduction to subst œÑ (N‚Ä≤ [ M‚Ä≤ ]).

Of course, if M ‚áõ M‚Ä≤, then subst-zero M pointwise parallel reduces to subst-zero M‚Ä≤.

       ‚Üí M ‚áõ M‚Ä≤
       ‚Üí par-subst (subst-zero M) (subst-zero M‚Ä≤)

We conclude this section with the desired corollary, that substitution respects parallel reduction.

Parallel reduction satisfies the diamond property

The heart of the confluence proof is made of stone, or rather, of diamond! We show that parallel reduction satisfies the diamond property: that if M ‚áõ N and M ‚áõ N‚Ä≤, then N ‚áõ L and N‚Ä≤ ‚áõ L for some L. The proof is relatively easy; it is parallel reduction‚Äôs raison d‚Äôetre.

  ‚Üí M ‚áõ N‚Ä≤

        ‚ü® N‚ÇÉ [ M‚ÇÉ ] , ‚ü® sub-par p5 p7 , sub-par p6 p8 ‚ü© ‚ü©
